Exmaple:    my_str_1 = 'Hello'
            my_str_2 = "World"

=> Multi-line string, you can use triple double quotes or single quotes:
Example:        my_str_3 = """Multiline
                string"""
                my_str_4 = '''Another
                multiline
                string'''

=> string contains single quotes, use double quotes to wrap the string, and vice versa:
quote = 'She said, "Hello World!"'
print(quote) # Output: She said, "Hello World!"

=> Python provides the "in" operator, to check if a string contains one or more characters, 
    which returns a boolean that specifies whether the character or characters exist in String or not.

=> length of a string and work with the individual characters in a string, a process called "indexing". 
    To get the length of a string, you can use the built-in len() function.

=> Each character in a string has a position called an "index". The index is zero-based, 
    meaning that the index of the first character of a string is 0, 
                    the index of the second character is 1, and so on.
    To access a character by its index, you use square brackets ([]) with the index of the character
    you want to access inside.

    "Negative indexing" is also allowed, so you can get the last character of any string with -1, 
        the second-to-last character with -2, and so on

=> "Primitive types" are simple and immutable, meaning they can't be changed once declared. 
=> "Reference types" can hold multiple values, and are either mutable or immutable.

Instead, all data gets treated as objects, and some objects are immutable while others are mutable.

=> "Immutable data types" can't be modified or altered once they're declared. 
    You can point their variables at something new, which is called "reassignment", 
    but you can't change the original object itself by adding, removing, or replacing any of 
    its elements.

=> You can combine multiple strings together with the plus (+) operator. 
    This process is called "string concatenation".
    But it works only with Strings. For example: If you are trying to concatenate a string with 
        a number, get a TypeError. This happens b/s Python doesn't automatically convert other data types
        like integers into strings to concatenate. IT requires all elements to be string.
    Use built-in function "str()" to convert number into string.

    You can also use the augmented assignment operator for concatenation. 
    This is represented by a plus and equals sign (+=), and performs both 
    concatenation and assignment in one step.

=> The process of inserting variables and expressions into a string is called "string interpolation". 
    Python has a category of string called "f-strings" (short for formatted string literals), 
        which allows you to handle interpolation with a compact and readable syntax.
    "F-strings" start with f (either lowercase or uppercase) before the quotes, and allow you to 
        embed variables or expressions inside replacement fields indicated by curly braces ({}).

=> "String slicing" lets you extract a portion of a string or work with only a specific part of it.
    Syntax: string[start:stop] / string[start:stop:step]
    Example:    my_str = 'Hello World'
                print(my_str[0: 7]) # H to W 
    Python will default to 0 or the end of the string - [:7] / [7:]
    stop index is non-inclusive, so [0:7] just extracted the characters from index 0, and up to, 
    but not including, the character at index 7.

    Apart from the start and stop indices, there's also an optional step parameter, 
    which is used to specify the increment between each index in the slice.

=> Built-in methods
    upper(): Returns a new string with all characters converted to uppercase.
    isupper(): Returns True if all letters in the string are uppercase and False if not.
    lower(): Returns a new string with all characters converted to lowercase.
    islower(): Returns True if all letters in the string are lowercase and False if not.
    strip(): Returns a new string with the specified leading and trailing characters removed. 
                If no argument is passed it removes leading and trailing whitespace.
    replace(old, new): Returns a new string with all occurrences of old replaced by new.
    split(separator): Splits a string on a specified separator into a list of strings. 
                        If no separator is specified, it splits on whitespace.
    join(iterable): Joins elements of an iterable into a string with a separator.
    startswith(prefix): Returns a boolean indicating if a string starts with the specified prefix.
    endswith(suffix): Returns a boolean indicating if a string ends with the specified suffix.
    find(substring): Returns the index of the first occurrence of substring, or -1 if it doesn't find one.
    count(substring): Returns the number of times a substring appears in a string.
    capitalize(): Returns a new string with the first character capitalized and the other characters lowercased.
    title(): Returns a new string with the first letter of each word capitalized.
