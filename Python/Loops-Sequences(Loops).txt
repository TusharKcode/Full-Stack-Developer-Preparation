"Loops" are used to repeat a block of code for a set number of times. 

=> "for" loop to iterate through a list and print each item to the console:
   - Also use a for loop to iterate through other iterables like a string.
Example: 
        programming_languages = ['Rust', 'Java', 'Python', 'C++']
        for language in programming_languages:
            print(language) # Rust Java Python C++
        
        for char in 'code':
            print(char) # c o d e

    - "nested" for loop:
    Example: 
            categories = ['Fruit', 'Vegetable']
            foods = ['Apple', 'Carrot', 'Banana']

            for category in categories:
                for food in foods:
                    print(category, food)

=> "while" loop is a loop will repeat a block of code until the condition is False. 
Example: 
        secret_number = 3
        guess = 0

        while guess != secret_number:
            guess = int(input('Guess the number (1-5): '))
            if guess != secret_number:
                print('Wrong! Try again.')

        print('You got it!')


=> The break statement is used to stop the execution of a loop. 
Example:
        developer_names = ['Jess', 'Naomi', 'Tom']
        for developer in developer_names:
            if developer == 'Naomi':
                break
            print(developer)

=> The continue statement is used to skip the current iteration of a loop and move onto the 
    next iteration.
Example:
        developer_names = ['Jess', 'Naomi', 'Tom']
        for developer in developer_names:
            if developer == 'Naomi':
                continue
            print(developer)

=> The "range()" function is used to generate a sequence of integers
    range() function only accepts integers as arguments, not floats.
    With the range() function is create a list of integers by using it with the list constructor. 
        The list constructor is used to convert an iterable into a list.
    Syntax: range(start, stop, step)
        - stop argument is non-inclusive.
    Example: 
            for num in range(2, 11, 2):
                print(num)

            numbers = list(range(2, 11, 2))
            print(numbers) # [2, 4, 6, 8, 10]

=> "enumerate()" function keeps track of the index for an iterable and returns an enumerate object.
    Each entry in the enumerate object (now a list) is a tuple containing a count, 
        followed by a value from the iterable passed to the enumerate() function.
    The enumerate() function also accepts an optional start argument that specifies the starting value
        for the count. If this argument is omitted, then the count will begin at 0.
